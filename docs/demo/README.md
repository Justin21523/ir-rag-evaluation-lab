# Demo Capture

This folder contains the Playwright verification assets for IR / RAG Evaluation Lab.

- `ir-rag-evaluation-lab-demo.mp4`: full guided walkthrough recording.
- `screenshots/`: viewport screenshots captured stage by stage.
- `playwright-report/`: HTML test report generated by Playwright.
- `playwright-artifacts/`: retained videos/traces from the latest capture run.

Run locally:

```bash
make refresh-lab
make text-mining DATASET_ID=sample_ir_demo_100
make api
cd frontend
VITE_PROXY_TARGET=http://127.0.0.1:8300 npm run dev -- --host 127.0.0.1 --port 5301
npm run capture:demo
```

The walkthrough covers the auto-start guided assistant, Pipeline Journey, Corpus, Experiment Workflow, Query Evaluator, Retrieval Comparison, Evaluation Analytics, RAG Citation Checker, Bad Cases, LLM Evaluation, Text Mining, Experiment Runs, and bilingual UI switching.
